# Microbit-Game-Suite
Game suite for the BBC Micro:bit.

Games included:
* Snake

Games in development:
* Dodge

Planned games:
* Pong
* Maze
* Space Invaders

Controls in all menus:
* A - select/confirm
* B - next/back

In-game controls should be intuitive.
